# Claude Code — Project Instructions

## Coolify CLI

The `coolify` binary is installed at `/opt/homebrew/opt/coolify-cli/bin/coolify` (also on PATH as `coolify`).

Use the Bash tool to run it directly. Always prefer `--format json` for output you need to parse.

### Reference docs

Coolify CLI docs are kept in `~/.claude/coolify-llms.txt` (global, available across projects).
